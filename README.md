## 프로젝트 목표

식물 관리, 커뮤니티, 분양, 일정 관리, AI 기반 식물 도감을 통합한 **식물 라이프스타일 플랫폼**입니다.

- **Frontend**: React, TypeScript, Vite, TailwindCSS, react-konva
- **Backend**: Spring Boot, JPA, QueryDSL
- **Database**: MySQL / PostgreSQL, Redis
- **Messaging**: RabbitMQ, STOMP
- **AI / API**: Pl@ntNet API, Gemini API
- **Map**: Kakao Map API, Naver Map API,
- **Infra**: Docker, Cloud Run / Firebase Hosting
